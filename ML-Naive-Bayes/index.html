朴素贝叶斯分类器

根据贝叶斯定理，后验概率可以写成

$$\begin{align} P(Y = c_k \mid X = x) & = \frac{P(Y = c_k) P(X = x \mid Y = c_k)}{P(X = x)} \ & = \frac{P(Y = c_k) P(X = x \mid Y = c_k)}{\sum_{k} P(Y = c_k) P(X = x \mid Y = c_k)} \end{align} \tag{1}$$

因此朴素贝叶斯分类器可以表示为

$$y = \arg \max_{c_k} \frac{P(Y = c_k) \prod_{j} P(X^{(j)} = x^{(j)} \mid Y = c_k)}{\sum_{k} P(Y = c_k) \prod_{j} P(X^{(j)} = x^{(j)} \mid Y = c_k)} \tag{2}​$$

其中，$X^{(j)}$表示第j个属性。由于，公式(2)中分母对于所有$c_k$都相同，所以

$$y = \arg \max_{c_k} P(Y = c_k) \prod_{j} P(X^{(j)} = x^{(j)} \mid Y = c_k) \tag{3}$$

对于连续属性，我们考虑猜测属性服从的概率分布，并根据概率分布给出概率。例如，假定第i个属性服从高斯分布，则

$$p(X^{(j)} = x^{(j)} \mid Y = c_k) = \frac{1}{\sqrt{2 \pi} \sigma_{c_k,j}} \exp(- \frac{(x^{(j)} - \mu_{c_k,j})^2}{2 \sigma_{c_k,j}^2}) \tag{4}$$

其中，$\mu_{c_k,j}$和$\sigma_{c_k,j}$分别是第$c_k$类样本在第j个属性上的取值的均值和方差。

参数估计

在朴素贝叶斯里的参数估计是通过最大似然估计法得到的。先验概率$P(Y = c_k)$的最大似然估计是

$$P(Y = c_k) = \frac{\sum_{i=1}^{N} I(y_i = c_k)}{N} \tag{5}$$

其中，$I(*)$是指示函数。条件概率$P(X^{(j)} = x^{(j)} \mid Y = c_k)$的最大似然估计是

$$P(X^{(j)} = x^{(j)} \mid Y = c_k) = \frac{\sum_{i=1}^{N} I(x_i^{(j)} = a_{jl},y_i = c_k)}{\sum_{i=1}^{N} I(y_i = c_k)} \tag{6}$$

其中，$a_{jl}$是第j个特征的可能取的第l个值。

拉普拉斯修正

对于某些没有样本的属性值，它们的概率为0，会使最后的乘积为0。所以，我们通过拉普拉斯修正，平滑概率值。

$$P(Y = c_k) = \frac{\mid D_{c_k} \mid + 1}{\mid D \mid + N} \tag{7}$$

$$P(X^{(j)} = x^{(j)} \mid Y = c_k) = \frac{\mid D_{c_k,x^{(j)}} \mid + 1}{\mid D_c \mid + N_j} \tag{8}$$

其中，$N$表示训练集D中可能的类别数，$N_j$表示第j个属性的可能的取值数量。
