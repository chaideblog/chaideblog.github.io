原问题

一个标准的优化问题的形式如下：

$$\min f_0 (x)$$

$$s.t.\ f_i (x) \leq 0$$

$$\ \ \ \ \ \ \ h_i (x) = 0$$

需要说明的是，该问题不要求是一个凸优化问题。Lagrange Dual的思想是在目标函数中考虑约束问题，添加约束条件的加权和，得到增广的目标函数。Lagrange对偶问题的形式如下：

$$L(x,\lambda,v)=f_0 (x) + \sum_{i=1}^m \lambda_i f_i(x) + \sum_{i=1}^{p} v_i h_i(x)$$

对偶问题

求解$$L(x,\lambda,v)$$的最小值，即

$$g(\lambda,v) = \inf_{x \in D} L(x,\lambda,v) = \inf_{x \in D} (f_0 (x) + \sum_{i=1}^m \lambda_i f_i(x) + \sum_{i=1}^{p} v_i h_i(x))$$

因为对偶函数是一族关于$$(\lambda,v)$$的仿射函数的逐点下确界，所以原问题不是凸的，对偶函数也是凹的。

对偶问题构成了原问题最优值$$p^*$$的下界，证明：

$$\sum_{i=1}^m \lambda_i f_i(x) + \sum_{i=1}^{p} v_i h_i(x) \leq 0$$

$$L(\tilde{x},\lambda,v)=f_0 (\tilde{x}) + \sum_{i=1}^m \lambda_i f_i(\tilde{x}) + \sum_{i=1}^{p} v_i h_i(\tilde{x}) \leq f_0(\tilde{x})$$

$$g(\lambda,v) \leq L(\tilde{x},\lambda,v) \leq f_0(\tilde{x})$$

其中，$$\tilde{x}$$是原问题的一个可行点。

求解对偶问题

Lagrange对偶函数给出了原问题的最优值$$p^*$$的下界，最好的下界：

$$\max g(\lambda,v)$$

$$s.t.\ \lambda \geq 0$$

Lagrange对偶问题是一个凸优化问题，因为极大化的目标函数是凹函数，且约束集是凸集合。因此，对偶问题的凸性和原问题是否是凸优化问题无关。

弱对偶性

Lagrange对偶问题的最优值$$d^$$，则有$$d^ \leq p^$$。即使原问题不是凸问题，$$d^ \leq p^$$亦成立，称为弱对偶性。同时，$$p^ - d^*$$称为最优对偶间隙。

强对偶性

若$$d^ = p^$$成立，强对偶性成立。一般条件下，强对偶性不成立，但$$f_0,f_1,…,f_m$$是凸函数，强对偶性通常成立。

强对偶性成立的条件

Slater条件

存在$$x \in relint\ D,$$满足$$f_i(x)<0,i=1,…,m$$。

当Slater条件成立且原函数凸，强对偶性成立。

KKT条件

假设$$f_0,f_1,…,f_m,h_1,…,h_p$$可微，并不假设这些函数是凸函数。

非凸函数的KKT

$$\nabla f_0(x^)+ \sum_{i=1}^m \lambda_i^ \nabla f_i(x^) + \sum_{i=1}^{p} v_i^ \nabla h_i(x^*) = 0$$

若强对偶成立，原问题最优解和对偶问题毕满足KKT条件。

凸函数的KKT

当原问题是凸函数，满足KKT条件的点是原函数和对偶函数的最优解。

$$\begin{cases} f_i(\tilde{x}) \leq 0, & i,=1,…,m \ h_i(\tilde{x})=0, & i=1,…,p \ \tilde{\lambda_i} \geq 0, & i=1,…,m \ \tilde{\lambda_i} f_i(\tilde{x})=0, & i=1,…,m \end{cases}$$

$$\nabla f_0(x^)+ \sum_{i=1}^m \lambda_i^ \nabla f_i(x^) + \sum_{i=1}^{p} v_i^ \nabla h_i(x^*) = 0$$

那么，$$\tilde{x}$$和$$(\tilde{\lambda}, \tilde{v})$$分别为原问题、对偶函数的最优解，对偶间隙为0，即

$$\begin{split} g(\tilde{\lambda}, \tilde{v}) &= L(\tilde{x},\tilde{\lambda}, \tilde{v}) \ &=f_0(\tilde{x})+ \sum_{i=1}^m \tilde{\lambda_i} f_i(\tilde{x}) + \sum_{i=1}^{p} \tilde{v_i} h_i(\tilde{x}) \ &=f_0(\tilde{x}) \end{split}$$
