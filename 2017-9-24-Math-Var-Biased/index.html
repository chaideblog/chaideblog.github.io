有偏估计&无偏估计

在统计学中，估计量的偏差（或偏差函数）是此估计量的期望值与估计参数的真值之差。偏差为零的估计量或决策规则称为无偏的。否则该估计量是有偏的。

均值是无偏的

$$\begin{align} \mathbb{E} [\hat{\mu}] &= \mathbb{E} [\frac{\sum_{n=1}^N x_n}{N}] \ &= \frac{1}{N} \mathbb{E} [\sum_{n=1}^N x_n] \end{align} \tag{1}$$

因为，$x_n$是独立同分布的。所以

$$\begin{align} \mathbb{E} [\hat{\mu}] &= \frac{1}{N} \sum_{n=1}^{N} \mathbb{E} [x_n] \ &= \frac{1}{N} N \mathbb{E} [x_n] \ &= \mu \end{align} \tag{2}$$

方差是有偏的

因为$\hat{\sigma^2} = \frac{1}{N} \sum_{1}^{N} (x_n - \hat{\mu})^2$，因此

$$\begin{align} \mathbb{E} [\hat{\sigma}^2] &= \frac{1}{N} \mathbb{E} [\sum_{n=1}^{N} (x_n - \hat{\mu})^2] \ &= \frac{1}{N} \mathbb{E} [\sum_{n=1}^{N} (x_n^2 - 2 \hat{\mu} x_n + \hat{\mu}^2)] \ &= \frac{1}{N} \mathbb{E} [\sum_{n=1}^{N} x_n^2 - 2 \sum_{n=1}^{N} \hat{\mu} x_n + \sum_{n=1}^{N} \hat{\mu}^2] \end{align} \tag{3}$$

因为，$\sum_{n=1}^{N} x_n = \hat{\mu} N$，所以

$$\begin{align} \mathbb{E} [\hat{\sigma}^2] &= \frac{1}{N} \mathbb{E} [\sum_{n=1}^{N} x_n^2 - N \hat{\mu}^2] \ &= \frac{1}{N} \mathbb{E} [\sum_{n=1}^{N} x_n^2] - \mathbb{E} [\hat{\mu}^2] \end{align} \tag{4}$$

又因为$x_n$是独立同分布的。所以

$$\mathbb{E} [\hat{\sigma}^2] = \mathbb{E} [x^2] - \mathbb{E} [\hat{\mu}^2] \tag{5}$$

我们又知道

$$\sigma^2 = \mathbb{E} [x^2] - \mathbb{E}^2 [x] \tag{6}$$

$$\sigma_{\hat{\mu}}^2 = \mathbb{E} [\hat{\mu}^2] - \mathbb{E}^2 [\hat{\mu}] \tag{7}$$

其中，$\sigma_{\hat{\mu}}^2$表示均值$\hat{\mu}$的方差。

$$\mathbb{E} [\hat{\sigma}^2] = (\sigma^2 + \mu^2) - (\sigma_{\hat{\mu}}^2 + \mu^2) = \sigma^2 - \sigma_{\hat{\mu}}^2 \tag{8}$$

$$\begin{align} \sigma_{\hat{\mu}}^2 &= VAR[\hat{\mu}] = VAR[\frac{1}{N} \sum_{n=1}^N x_n] \ &= \frac{1}{N^2} VAR[\sum_{n=1}^{N} x_n] = \frac{1}{N} N \sigma^2 \end{align} \tag{9}$$

因此，

$$\mathbb{E} [\hat{\sigma}^2] = \sigma^2 - \frac{1}{N} \sigma^2 = \frac{N-1}{N} \sigma^2 \tag{10}$$
