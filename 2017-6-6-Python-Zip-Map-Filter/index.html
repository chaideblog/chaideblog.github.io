zip、map和filter函数创建相应的迭代器。

zip函数

zip()是Python的内建函数，接收一系列可迭代的对象，打包成zip对象。

当参数长度不同时，zip会以最短序列的长度来截取得到元组。

map函数

map()是Python的内建函数，形式为map(function, object)，map函数将object按照function函数转化。

map函数还有其他高级操作，可以将多个序列作为参数。

map函数比列表解析运行的速度更快，而列表解析的运行速度比for循环快。此外，map函数和列表迭代编写的代码量更少。

filter函数

filter函数用于过滤出一些元素，返回可迭代对象。
