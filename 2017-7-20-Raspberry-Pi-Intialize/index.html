中文化

命令行终端执行sudo raspi-config，选择4 Localisation Options Set up language and regional settings to match your location，选择I1 Change Locale Set up language and regional settings to match your location。

去掉en_GB.UTF-8 UTF-8，勾上：“en_US.UTF-8 UTF-8”、“zh_CN.UTF-8 UTF-8”、“zh_CN.GBK GBK”。（操作提示：按空格键在前面打勾或去掉勾（星号=勾），PageUp PageDown快速翻页，Tab键跳到OK按钮上）下一屏幕默认语言选zh_CN.UTF-8。

重新启动后，系统支持中文。
