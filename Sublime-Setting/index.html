通过Package Control 来安装插件：

  1. 按下Shift + Command + P调出命令面板。
  2. 输入install调出Package Control: Install Package选项，按下回车。
  3. 输入插件名称并回车，稍等几秒就安装好了，有的插件可能需要重启Sublime Text才能激活。

下面是一些我配置的常规插件。

  • ConvertToUTF8通过本插件，您可以编辑并保存目前编码不被 Sublime Text 支持的文件，特别是中日韩用户使用的 GB2312，GBK，BIG5，EUC-KR，EUC-JP 等。ConvertToUTF8 同时支持 Sublime Text 2 和 3。
  • Terminal这个插件可以让你在Sublime中直接使用终端打开你的项目文件夹，并支持使用快捷键Control + Shift + T。
