面试过程

搜狐只面了一面，但是一面大约花了1个半小时，拿到offer。比我来的晚的面试的同学走的也比我早，但是我觉得面试官想多了解了解你是一件好事情。

算法题

上来给了张纸，上面有4道算法题，任选2道题。

  1. 3Sum(Medium)

    给定$$n$$个整数的数组nums，是否有元素$$a,b,c$$在nums中使$$a + b + c = 0$$？ 找到数组中所有不重复的三元组，它们的总和为零。

  2. Unique Paths II(Medium)

    机器人位于$$m \times n$$网格的左上角（在下图中标记为“开始”）。机器人只能在任何时间点向下或向右移动。 机器人正试图到达网格的右下角（在下图中标记为“完成”）。现在考虑是否在网格中添加了一些障碍。 有多少独特的路径？

  3. Container With Most Water(Medium)

    给定$$n$$个非负整数$$a_1, a_2, \cdots, a_n$$，其中每个表示坐标$$(i, a_i)$$处的点。 绘制$$n$$条垂直线，使得线$$i$$的两个端点位于$$(i, a_i)$$和$$(i, 0)$$。 找到两条线，它们与$$x$$轴一起形成一个容器，这样容器就含有最多的水。

  4. Linked List Cycle(Easy)

    给定一个链表，确定它是否有一个循环。为了表示给定链表中的循环，我们使用整数pos表示环的入口到的链表0索引的相对位置。 如果pos为-1，则链表中没有循环。

我选了2和3，然后面试官改进了第3题。

  1. Trapping Rain Water(Hard)

    给定$$n$$个非负整数表示每个柱的宽度为1的高程图，计算下雨后能够捕获的水量。

知识点

  1. 香农公式与熵的关系，熵、条件熵、交叉熵的互相转化？
  2. LR的损失函数，LR中概率形式的推导？
  3. 最大似然估计和最大后验估计的形式？
  4. XGBoost分裂原理？对缺失值敏感吗？如何处理缺失值？
  5. 正则化的方式？
  6. 批正则原理和公式？为什么公式这样子？

心得

基础知识很重要。
