更改目录索引路径

如果想更改目录索引的目录路径，可以更改/etc/apache2/site-available目录下的000-default.conf文件。000-default.conf文件是默认的虚拟主机配置文件，修改其中的DocumentRoot参数，可以更改目录索引的路径。

修改后，使用a2enconf 000-default.conf命令使修改的配置工作。该命令实际上是在／etc/apache2/site-enable目录下软链接/etc/apache2/site-available目录下的000-default.conf文件。

最后，service apache2 restart重启Apache服务。

重启Apache服务，下面介绍上述配置中的几个参数：

  1. Options Indexes：开启目录索引功能。
  2. IndexStyleSheet “./style.css”：指定修改Apache默认的目录索引样式。
  3. IndexOptions：该参数下的选项非常多，SuppressHTMLPreamble规定使用用户自己定义的部分，而不是Apache默认的；NameWidth文件命显示字节数；DescriptionWidth描述显示字节数；FoldersFirst目录优先；HTMLTable允许HTML格式。
  4. IndexIgnore：该参数设定Apache需要忽略的文件，该参数设定的文件不会显示在网页中。
  5. HeaderName和ReadmeName：分别定义了索引头部嵌入文件和索引底部嵌入文件，下面会详细介绍。
